Módulos iniciales

Estos 3 modulos están disponibles para el uso y optimizado de acuerdo a problemáticas o casos reales.

Resuelve eficientemente el tema de información o base de datos, gestiona con agilidad la cobranza y mantiene registros en tiempo real cada proceso que se realiza en la garita.

Residentes

Toda una base de datos personalizados y con filtros detallados.

Contabilidad

Garita

Reporte de todos los eventos que ocurre en la garita.
